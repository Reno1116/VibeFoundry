# 常用任务模板

## 整理 brief

```text
请先按 UTF-8 读取所有中文文档。
我想把一个想法整理成可执行任务，请按仓库 brief 模板帮我输出，并明确目标、范围、约束、成功标准是否齐全。
```

## 输出 spec

```text
请先按 UTF-8 读取所有中文文档。
我已经有目标、范围、约束和成功标准，请按仓库 spec 模板整理实现方案，至少写清输入输出、边界和验证方式。
```

## 进入 build

```text
请先按 UTF-8 读取所有中文文档。
请检查现有 brief 和 spec 是否足够开始实现；如果足够，先提醒我做人工确认，再给出最小可运行版本的落地步骤。
```

## 补 validate

```text
请先按 UTF-8 读取所有中文文档。
请根据现有实现和测试结果，按验收模板帮我补一份最小 validate 记录，至少说明结果、验证方式和缺口。
```
